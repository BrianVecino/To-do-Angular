✅ To-Do App — Angular
Aplicación de gestión de tareas desarrollada con Angular 21 y Signals. Permite crear, completar y eliminar tareas con persistencia en LocalStorage.

🔗 Demo
👉 brianvecino.github.io/To-do-Angular

✨ Funcionalidades

➕ Crear nuevas tareas.
✔️ Marcar tareas como completadas.
🗑️ Eliminar tareas.
💾 Persistencia en LocalStorage (las tareas se mantienen al recargar).

🛠️ Tecnologías

Angular 21 (standalone components).
Signals — estado reactivo sin NgRx.
TypeScript.
CSS3.
LocalStorage — persistencia sin backend.

🚀 Instalación y uso
Requisitos: Node.js 18+Angular CLI 21.

Pasos

Cloná el repositorio:

bash   git clone https://github.com/BrianVecino/To-do-Angular.git
   cd To-do-Angular

Instalá las dependencias:

bash   npm install

Levantá el servidor de desarrollo:

bash   ng serve

Abrí el navegador en http://localhost:4200
